Nyiss meg egy terminált, navigálj ahhoz a könyvtárhoz, ahol a Python szkript és a YAML fájl található, majd futtasd:

Bash
python run_simulation.py
 Ha más néven mentetted a YAML fájlt, vagy máshol van, megadhatod a -c kapcsolóval:

Bash
python run_simulation.py -c /elérési/út/config_uj.yaml
 Ha a C futtatható fájl máshol van, a -e kapcsolóval adhatod meg:

Bash
python run_simulation.py -e /masik/utvonal/a/simulation_programhoz
